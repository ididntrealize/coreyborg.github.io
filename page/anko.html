My contribution

I worked with Anko as their primary eCommerce designer and manager for 11 months in 2018. I transformed their blank slate template into a high performance ecommerce machine, performing A/B tests and incorporating analytic feedback into future designs. I was very active in Google Analytics and Google Optimize to inform future directions to take the site (and keep key stakeholders apprised). I also took on reviewing and incorporating direct user feedback into the site from screen share sessions performed by users on usertesting.com .

I also implemented an all new checkout page in react with custom functionality requested by shareholders. The checkout page is the most important step of the conversion funnel, and you need to be positive that all itâ€™s functionality works for every release, so we created unit tests. British English Translator

Anko is an Australian brand, so their product databases are filled with British spellings (like colour, or realise). I wrote this script to americanize the spelling and units for their products.

View source on Github 
function translateEnglish(fullText) {
    //preserve original text to test against updated string
    let newStr = fullText;
    
    //split all words into an array, discounting HTML tags
    let tagsToSpaces = fullText.replace(/(<([^>]+)>)/ig, " "),
        wordsToCheck = tagsToSpaces.split(' ');
    
    //check each word in passed string with the spelling diff lookup table
    wordsToCheck.forEach(function (elem) {
        let formatted = elem.toLowerCase();
        formatted = formatted.replace(/[^a-zA-Z ]/g, "");
    
        if (spellingDifferences[formatted]) {
    
            let anythingRevomed = elem.replace(/[a-zA-Z ]/g, "");
    
            //AUS English word found, replace it (with any random formatting associated)
            newStr = newStr.replace(elem, spellingDifferences[formatted] + anythingRevomed);
    
            if (debug) { console.log(elem + ' => ' + spellingDifferences[formatted] + anythingRevomed) }
    
        }
    });
    
    return newStr;
    
    }
